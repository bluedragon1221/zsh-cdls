# Original README
## CDLS
List contents of a directory upon CD-ing to that directory.

Installable with:
- [Zplug](https://github.com/zplug/zplug): `zplug "rummik/zsh-cdls"`
- [Antigen](https://github.com/zsh-users/antigen): `antigen bundle rummik/zsh-cdls`

# New README
